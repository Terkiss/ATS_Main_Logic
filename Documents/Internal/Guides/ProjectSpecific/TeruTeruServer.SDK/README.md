# 🏗️ 기초 공사 파트 (TeruTeruServer.SDK)

"튼튼한 건물을 지으려면 좋은 재료가 필요해요!" ✨
이 구역은 우리 놀이공원의 모든 곳에서 공통으로 사용하는 '표준 재료'들이 모여 있는 곳이에요.

## 🏢 이곳은 무엇을 하는 곳인가요?

이곳은 **SDK(Software Development Kit)**라고 불러요. 
놀이공원(서버)을 만들 때 필요한 **약속(인터페이스)**과 **공통 도구(유틸리티)**들을 정의해 놓은 곳이죠. 
예를 들어, "손님에게 인사할 때는 '안녕하세요'라고 말하자!"라는 약속을 여기서 정해두면, 모든 직원이 똑같이 인사할 수 있게 돼요.

## 🗺️ 이 방의 지도 (폴더 구조)

*   **Enums**: 🏷️ 서버에서 사용하는 '이름표' 모음이에요. (예: `ProtocolSelect.cs` - 어떤 대화를 할지 정하는 번호표)
*   **Protocol**: 📝 서버와 손님이 주고받는 '편지 양식'들이 들어있어요.
*   **Interfaces**: 🤝 "우리 이런 기능을 만들기로 약속하자!"라는 설계도들이 모여 있어요.
*   **Util**: 🛠️ 어디서든 쓸 수 있는 만능 도구함이에요. (예: 로그를 남기는 `TeruTeruLogger.cs`)

## 🔍 가장 중요한 코드 읽어보기

가장 많이 보게 될 파일은 **`Enums/ProtocolSelect.cs`** 파일이에요.

*   **무슨 일을 하나요?**: 서버와 클라이언트가 서로 어떤 종류의 대화를 나누고 있는지 숫자로 알려줘요. 
*   예를 들어, `LoginProtocol = 1`이라면 "아, 지금 1번 편지가 왔으니 로그인하려나 보구나!" 하고 알 수 있는 거죠.

## 🪄 나만의 마법 부리기 (따라하기)

우리가 새로운 대화 주제를 추가해볼까요?

1.  `TeruTeruServer.SDK/Enums/ProtocolSelect.cs` 파일을 열어보세요.
2.  중간에 `MyMagicProtocol = 999,` 라고 한 줄을 추가해보세요.
3.  이제 우리 서버는 999번이라는 새로운 대화 주제를 이해할 준비가 된 거예요!
