# 🧰 TeruTeruServer.SDK (만능 도구상자)

안녕하세요! 이곳은 놀이공원을 관리하는 직원(개발자)들을 위한 **'만능 도구상자'** 방이에요. ✨

## 🎈 이곳은 무엇을 하는 곳인가요?
SDK는 'Software Development Kit'의 약자인데, 쉽게 말해서 **'놀이공원 관리를 쉽게 해주는 도구 모음집'**이라고 생각하면 돼요. 
놀이공원 관제탑이나 놀이기구 등 여러 곳에서 공통으로 쓰는 부품들이나 약속들이 이곳에 모두 모여 있답니다.

## 🗺️ 이 방의 지도 (폴더 구조)
이 도구상자 방에는 용도별로 서랍들이 나뉘어 있어요.

* 🏷️ **`Enums` 폴더**: 놀이공원의 **'이름표 보관소'**예요. 컴퓨터는 숫자(1, 2, 3...)밖에 모르기 때문에, "1번은 로그인, 2번은 연결"처럼 숫자에 예쁜 이름표를 붙여주는 곳이랍니다.
* 📝 **`Interfaces` 폴더**: 놀이공원 직원들의 **'업무 지침서'**예요. "안내원이라면 반드시 인사하기 기능을 가지고 있어야 해!" 라고 필수 규칙들을 정해둔 곳이에요.
* ✉️ **`Protocol` 폴더**: 손님들과 주고받는 **'표준 편지봉투'**들이 모여 있는 곳이에요. 이 봉투에 담지 않으면 편지(데이터)를 보낼 수 없어요.
* 🛠️ **`Util` 폴더**: 암호를 풀어주는 기계, 시간을 재는 타이머 같은 **'편리한 도구'**들이 들어있는 서랍이에요.
