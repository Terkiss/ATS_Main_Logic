# 🗼 TeruTeruServer.Runtime (놀이공원 관제탑)

안녕하세요! 이곳은 우리 놀이공원에서 가장 중요한 **'관제탑'**이에요. ✨

## 🎈 이곳은 무엇을 하는 곳인가요?
놀이공원에 수많은 손님(클라이언트)들이 놀러 오면, 가장 먼저 맞이하는 곳이 바로 이곳이에요.
손님들이 입구에 오면 문을 열어주고(네트워크 연결), 손님들이 보낸 쪽지(패킷)를 받아서 안전한지 검사한 다음, 알맞은 놀이기구로 안내해 주는 역할을 한답니다.

## 🗺️ 이 방의 지도 (폴더 구조)
이 관제탑 방(폴더) 안에는 여러 가지 중요한 서랍과 장치들이 있어요.

* 🖥️ **`MainServer.cs`**: 관제탑의 메인 컴퓨터예요! 손님들이 들어오고 나가는 것을 전부 지켜보고 관리해요.
* 🛡️ **`Pipeline` 폴더**: 손님들이 보낸 쪽지(패킷)를 검사하는 **'보안 검색대'**예요. 이상한 쪽지는 없는지, 암호는 잘 걸려 있는지 확인하죠.
* 📖 **`DB` 폴더**: 우리 놀이공원의 **'회원 명부 보관소'**예요. 누가 언제 왔었는지 기록을 남기는 곳이랍니다.
* 🧩 **`PluginManager.cs`**: 놀이공원에 새로운 이벤트 부스를 쉽게 추가하고 뺄 수 있게 도와주는 마법의 장치예요.
