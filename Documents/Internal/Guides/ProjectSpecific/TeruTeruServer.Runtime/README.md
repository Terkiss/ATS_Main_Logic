# ❤️ 엔진/심장 파트 (TeruTeruServer.Runtime)

"놀이공원에 불을 밝히고 기계를 돌리는 핵심 장치예요!" ⚡
이 구역은 우리 서버가 실제로 '살아서 움직이게' 만드는 아주 중요한 곳이에요.

## 🏭 이곳은 무엇을 하는 곳인가요?

이곳은 **런타임(Runtime)**이라고 불러요. 
컴퓨터가 서버 코드를 읽어서 실제로 손님의 접속을 기다리고(Listen), 데이터를 주고받는(Network I/O) 모든 과정을 담당하죠. 
마치 놀이공원의 거대한 발전소나 제어실 같은 곳이랍니다!

## 🗺️ 이 방의 지도 (폴더 구조)

*   **MainServer.cs**: 🏰 놀이공원의 정문이에요! 손님들이 여기서 줄을 서고 들어와요.
*   **GameEngine**: ⚙️ 놀이공원의 시계추 같은 곳이에요. (예: `GameLoop.cs` - 매초 20번씩 "째깍째깍" 소리를 내며 게임을 진행시켜요)
*   **Pipeline**: 🛡️ 손님이 들고 오는 가방을 검사하는 보안 검색대예요. (나쁜 데이터가 들어오지 못하게 막아줘요!)
*   **Clustering**: 🌐 놀이공원이 너무 커져서 옆 동네(다른 서버)와 연결해야 할 때 도와줘요.

## 🔍 가장 중요한 코드 읽어보기

가장 핵심이 되는 파일은 **`MainServer.cs`**예요.

*   **무슨 일을 하나요?**: 109번째 줄에 있는 `StartServer()` 함수를 보세요. 여기가 바로 놀이공원의 '개장 선언'을 하는 곳이에요!
*   **어려운 용어**: **Socket(소켓)** - 서버와 손님 사이의 '전화선' 같은 거예요. 이 선을 통해 데이터를 주고받죠.

## 🪄 나만의 마법 부리기 (따라하기)

놀이공원 입구의 환영 메시지를 바꿔볼까요?

1.  `TeruTeruServer.Runtime/MainServer.cs` 파일의 137번째 줄 근처를 찾아보세요.
2.  `Console.WriteLine("Server Start");` 부분을 `"Welcome to TeruTeru World! ✨ Server is ready!"`로 바꿔보세요.
3.  서버를 켰을 때 검은 화면에 내가 적은 멋진 인사말이 나올 거예요!
