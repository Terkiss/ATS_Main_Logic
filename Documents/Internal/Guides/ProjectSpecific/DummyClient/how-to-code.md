# 🛠️ 로봇 손님 조종하기 (코드 수정법)

안녕하세요! 이번에는 로봇 손님의 행동 지침서인 `DummyClient/Program.cs` 파일을 열어서 코드를 조금 수정해 볼 거예요.
로봇이 관제탑에 보내는 쪽지의 내용을 우리가 원하는 대로 바꿔볼까요? ✨

## 🧐 가장 중요한 코드 읽어보기

테스트를 담당하는 `DummyClient/Program.cs` 파일을 열어주세요.

### 1. 놀이공원으로 출발! (`new TeruClient`)
파일의 **30번째 줄**을 보면 이렇게 적혀있어요.
```csharp
using var client = new TeruClient("127.0.0.1", 3000);
```
여기서 `127.0.0.1`은 **"내 컴퓨터 안에 있는 놀이공원"**이라는 뜻의 특별한 주소이고, `3000`은 놀이공원의 출입문 번호(Port)예요. 로봇은 이 주소를 보고 찾아간답니다.

### 2. 관제탑에 쪽지 보내기 (`InvokeRpcAsync`)
조금 더 아래로 내려가서 **51번째 줄** 쯤을 보면 서버에 쪽지를 보내는 부분이 있어요.
```csharp
await client.InvokeRpcAsync("Echo", "Hello Server");
```
관제탑 아저씨한테 "Echo(메아리)" 라는 마법을 써달라고 하면서, 내용으로는 "Hello Server"를 보내는 거예요.

## 🪄 나만의 마법 부리기 (따라하기)

로봇 손님이 딱딱하게 "Hello Server" 대신 귀엽게 인사를 하도록 만들어 볼까요?

1. `DummyClient/Program.cs` 파일을 열어주세요.
2. **51번째 줄** 근처에 있는 `InvokeRpcAsync` 부분을 찾으세요.
3. `"Hello Server"`라고 적힌 글자를 지우고, 여러분이 보내고 싶은 메시지로 바꿔보세요!
   ```csharp
   await client.InvokeRpcAsync("Echo", "💖 관제탑 아저씨! 로봇 손님 1호가 무사히 도착했어요! 💖");
   ```
4. 저장하고 서버(`Cli`)를 켠 다음, 로봇 손님(`DummyClient`)도 실행해 보세요. 서버 화면에 여러분의 귀여운 메시지가 뜰 거예요! 🎉

## ⚠️ 주의사항

로봇이 찾아가는 놀이공원 주소인 `127.0.0.1`과 `3000`이라는 숫자를 함부로 바꾸면 큰일 나요! 
이 주소가 틀려지면 로봇 손님은 놀이공원 입구를 찾지 못하고 우주 미아가 되어버린답니다. 주소 부분은 조심조심 눈으로만 봐주세요! 🤖
