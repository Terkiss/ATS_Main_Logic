# 🤖 DummyClient (로봇 손님 테스트 센터)

안녕하세요! 이곳은 놀이공원이 정식으로 문을 열기 전에, 우리 놀이기구들이 잘 작동하는지 확인하기 위해 **'로봇 손님'**을 보내보는 테스트 센터예요. ✨

## 🎈 이곳은 무엇을 하는 곳인가요?
관제탑 코드를 고치거나 새로운 놀이기구를 만들었을 때, 진짜 손님들을 부르기 전에 미리 안전 검사를 해야겠죠?
이 폴더에 있는 코드는 사람이 마우스나 키보드를 누르지 않아도, 자동으로 관제탑에 연결하고 로그인을 시도해보는 **'가짜 손님(Dummy Client)'** 역할을 한답니다.

## 🗺️ 이 방의 지도 (폴더 구조)
이 방에는 로봇 손님이 어떻게 행동해야 하는지 적혀 있는 지침서가 있어요.

* 📝 **`Program.cs`**: 로봇 손님 1호의 **'행동 지침서'**예요. "먼저 관제탑 입구로 가라!", "로그인을 시도해라!", "서버에 '안녕'이라고 쪽지를 보내봐라!" 같은 명령들이 순서대로 적혀 있답니다.
