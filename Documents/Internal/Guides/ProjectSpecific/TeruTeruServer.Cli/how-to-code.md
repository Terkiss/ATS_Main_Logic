# 🛠️ 관제탑 파트 코드 수정법 (CLI)

안녕! 여기서는 놀이공원의 '조종실' 설정을 어떻게 바꾸는지 배워볼 거야. ✨

## 📝 서버 설정값(Config) 바꾸기

우리 놀이공원의 정문을 몇 번 포트로 열지, 몇 명까지 받을지 결정하는 법이야.

### 1단계: 설정 메모장 열기
`TeruTeruServer.Cli/config.txt` 파일을 열어봐. (혹은 프로젝트 폴더 바로 아래에도 있을 수 있어!)

### 2단계: 값 수정하기
내용을 보면 아래와 비슷한 형식이 보일 거야.

```text
Port=3000
MaxConnection=1000
# 여기에 새로운 설정 추가하기!
ServerName=MyMagicWorld
```

### 3단계: 무엇이 변했나요?
이제 서버를 다시 켜면 내가 바꾼 설정값대로 놀이공원이 운영될 거야!

## 📝 서버가 시작될 때 하는 일 추가하기

서버가 켜지자마자 화면에 멋진 로고를 띄워보자!

### 1단계: 메인 컴퓨터 확인
`TeruTeruServer.Cli/Program.cs` 파일을 열어봐.

### 2단계: 코드 수정하기
58번째 줄 근처에 `Console.WriteLine("=== TeruTeruServer AI Engine Runtime Started ===");` 라는 줄을 찾아봐.

```csharp
// 58번째 줄 근처에 추가! ✨
Console.WriteLine("***********************************");
Console.WriteLine("*   WELCOME TO MY COOL SERVER!    *");
Console.WriteLine("***********************************");
```
