# 🔘 관제탑 명령어 세트 (TeruTeruServer.Commands)

"관제탑 조종석에 앉아서 누르는 반짝이는 버튼들이에요!" ✨
이 구역은 관리자가 서버를 운영할 때 사용하는 **각종 명령어**가 정의되어 있는 곳이에요.

## 🏢 이곳은 무엇을 하는 곳인가요?

이곳은 **커맨드 패턴(Command Pattern)**이 적용된 곳이에요. 
"서버 꺼!", "상태 보여줘!" 같은 각각의 명령을 하나의 독립된 '버튼(클래스)'으로 만들어 관리하죠. 
마치 TV 리모컨의 각 버튼이 각자 맡은 일(채널 변경, 볼륨 조절)을 하는 것과 같답니다.

## 🗺️ 이 방의 지도 (폴더 구조)

*   **CommandHandler.cs**: 🧠 리모컨의 본체예요. 내가 어떤 버튼을 눌렀는지 확인하고 알맞은 기능을 실행시켜주죠.
*   **ICommand.cs**: 📐 모든 버튼이 가져야 할 공통적인 모양(설계도)이에요. "모든 버튼은 '누르기(Execute)' 기능이 있어야 해!"라고 정해두는 거죠.
*   **HealthCommand.cs**: ❤️ 서버가 아픈 곳은 없는지, 전기는 잘 들어오는지 체크하는 '검진 버튼'이에요.
*   **ExitCommand.cs**: 🛑 놀이공원의 문을 안전하게 닫는 '종료 버튼'이에요.

## 🔍 가장 중요한 코드 읽어보기

가장 핵심이 되는 파일은 **`CommandHandler.cs`**예요.

*   **무슨 일을 하나요?**: 19번째 줄부터 24번째 줄을 보세요. 여기서 리모컨에 어떤 버튼들이 있는지 등록하고 있어요.
*   **어려운 용어**: **Execute(엑제큐트)** - '실행하다'라는 뜻이에요. 버튼을 눌렀을 때 실제로 일어나는 마법 같은 일들이 적혀 있는 함수 이름이죠.
