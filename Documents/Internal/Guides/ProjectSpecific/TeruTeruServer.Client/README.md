# 🎒 손님용 도구 세트 (TeruTeruServer.Client)

"놀이공원에 놀러 올 때 챙겨야 할 필수 아이템들이에요!" ✨
이 구역은 진짜 손님(클라이언트)이 서버에 접속할 때 사용하는 **편리한 도구들**이 모여 있는 곳이에요.

## 🏢 이곳은 무엇을 하는 곳인가요?

이곳은 **클라이언트 SDK(Client SDK)**라고 불러요. 
서버와 복잡하게 대화하는 과정을 아주 쉽게 만들어주죠. 
예를 들어, "서버에 연결해!"라는 복잡한 네트워크 명령어를 `ConnectAsync()`라는 아주 간단한 버튼 하나로 만들어주는 마법 같은 곳이랍니다.

## 🗺️ 이 방의 지도 (폴더 구조)

*   **TeruClient.cs**: 📱 손님이 들고 있는 '스마트폰 앱'의 본체와 같아요. 서버와 연결하고, 로그인하고, 대화하는 모든 중심이 되죠.
*   **ClientProtocolRouter.cs**: 📮 서버에서 온 편지를 읽고 "이건 누구한테 전해줄까?" 하고 알맞은 방으로 배달해주는 우체부 아저씨예요.
*   **P2PManager.cs**: 🤝 친구들과 직접 만나서 이야기하고 싶을 때(P2P 통신) 도와주는 가이드예요.

## 🔍 가장 중요한 코드 읽어보기

가장 핵심이 되는 파일은 **`TeruClient.cs`**예요.

*   **무슨 일을 하나요?**: 50번째 줄의 `ConnectAsync()` 함수를 보세요. 서버 주소를 보고 입구를 찾아가는 일을 해요.
*   **어려운 용어**: **JWT Token(JWT 토큰)** - 19번째 줄의 `_jwtToken`을 보세요. 로그인을 성공하면 서버가 주는 '자유이용권 팔찌' 같은 거예요. 이 팔찌가 있어야 다른 놀이기구를 탈 수 있답니다!

## 🪄 나만의 마법 부리기 (따라하기)

서버와 연결됐을 때 화면에 나오는 환영 메시지를 조금 더 친절하게 바꿔볼까요?

1.  `TeruTeruServer.Client/TeruClient.cs` 파일의 59번째 줄을 찾아보세요.
2.  `Log($"Connected to server at {_serverIp}:{_serverPort}");` 부분을 `Log($"와아! {_serverIp} 놀이공원에 무사히 도착했어요! ✨");`로 바꿔보세요.
3.  이제 손님이 서버에 접속하면 훨씬 더 기분 좋은 로그가 뜰 거예요!

## ⚠️ 주의사항

> **"팔찌(토큰)를 잃어버리면 아무것도 할 수 없어요!"** 🔑
>
> `_jwtToken`은 서버가 나를 믿어준다는 증거예요. 이 토큰이 없으면 서버는 내가 누구인지 몰라서 아무런 대답도 해주지 않는답니다. 로그인 과정이 잘 돌아가는지 항상 확인해야 해요!
