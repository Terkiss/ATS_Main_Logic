# 📱 TeruTeruServer.Client (손님용 무전기 세트)

안녕하세요! 이곳은 놀이공원 손님(클라이언트)들이 관제탑과 쉽게 연락할 수 있도록 도와주는 **'무전기 세트'** 방이에요. ✨

## 🎈 이곳은 무엇을 하는 곳인가요?
우리가 유니티(Unity) 같은 엔진으로 멋진 게임 화면을 만들었다고 해봐요. 게임 속 캐릭터가 움직이면, 그 정보를 관제탑(서버)에 알려줘야겠죠?
이 폴더는 게임을 만드는 개발자들이 복잡한 네트워크 코드를 몰라도, 아주 쉽게 "로그인 할게요!", "총 쏠게요!" 하고 관제탑에 쪽지를 보낼 수 있게 만들어둔 친절한 도우미예요.

## 🗺️ 이 방의 지도 (폴더 구조)
이 방에는 손님들이 사용할 다양한 통신 기기들이 준비되어 있어요.

* 📡 **`TeruClient.cs`**: 가장 중요한 **'메인 무전기'**예요! 관제탑과 연결하고, 쪽지를 보내고 받는 모든 일을 담당해요.
* 귓속말 **`P2PManager.cs`**: 손님들끼리 관제탑을 거치지 않고 직접 빠르게 통신할 수 있게 도와주는 **'비밀 통신 장치'**랍니다.
* 📬 **`ClientProtocolRouter.cs`**: 관제탑에서 답장(쪽지)이 오면, "이건 로그인 성공 쪽지네!", "이건 누가 움직였다는 쪽지네!" 하고 척척 분류해 주는 도우미예요.
