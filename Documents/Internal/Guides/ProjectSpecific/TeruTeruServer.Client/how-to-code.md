# 🛠️ 손님용 도구 세트 코드 수정법 (Client)

안녕! 여기서는 우리 손님들이 서버와 더 스마트하게 소통하는 법을 배워볼 거야. ✨

## 📝 타임아웃 시간 조절하기

서버에게 질문을 던졌는데 대답이 너무 늦게 오면, 손님은 언제까지 기다려야 할까? 이 시간을 조절해볼 거야.

### 1단계: 대기 시간 찾기
`TeruTeruServer.Client/TeruClient.cs` 파일을 열어봐.

### 2단계: 코드 수정하기
124번째 줄 근처를 보면 `Task.Delay(5000)` 이라는 숫자가 있어.

```csharp
// 124번째 줄 근처
// 5000은 5초를 의미해! ⏱️
var completedTask = await Task.WhenAny(tcs.Task, Task.Delay(5000));
```

### 3단계: 무엇이 변했나요?
만약 서버가 너무 느리다면 이 숫자를 `10000`(10초)으로 늘려줄 수 있어. 그럼 손님은 조금 더 느긋하게 서버의 대답을 기다려줄 거야.

## 👂 새로운 로그 기록 남기기

손님이 서버와 대화할 때마다 어떤 일이 일어나는지 더 자세히 기록해볼까?

### 1단계: 기록 방식 확인
`TeruClient.cs` 파일의 맨 아래쪽에 있는 `Log` 함수(278번째 줄)를 찾아봐.

### 2단계: 코드 수정하기
```csharp
private void Log(string message)
{
    // 메시지 앞에 귀여운 아이콘을 붙여보자! 🤖
    OnLog?.Invoke($"[손님용 스마트폰] 📱 {message}");
}
```
