# 🛠️ 무전기 사용법 (코드 수정법)

안녕하세요! 이번에는 손님들의 메인 무전기인 `TeruClient.cs` 파일을 열어서, 무전기가 어떻게 작동하는지 구경하고 작은 마법을 부려볼 거예요. ✨

## 🧐 가장 중요한 코드 읽어보기

클라이언트 쪽에서 가장 중요한 파일은 `TeruTeruServer.Client/TeruClient.cs` 예요.

### 1. 관제탑에 연결하기 (`ConnectAsync` 함수)
이 파일 안에는 `public async Task<bool> ConnectAsync(...)` 라는 함수가 있어요.
이 함수는 마치 놀이공원 입구에 도착해서 "저 들어가도 되나요?" 하고 문을 똑똑 두드리는 역할을 해요. 관제탑의 주소(IP)와 문 번호(Port)를 넣고 연락을 시도하죠.

### 2. 로그인 해보기 (`LoginAsync` 함수)
또 `public async Task<bool> LoginAsync(string userId, string password)` 라는 함수도 있어요.
이건 관제탑과 연결이 된 후에, "제 아이디는 이거고요, 비밀번호는 이거예요!" 하고 로그인 쪽지를 보내는 곳이에요. 만약 로그인이 성공하면, 관제탑에서 아주 중요한 **자유이용권(AuthToken)**을 발급해 준답니다.

---

## 🪄 나만의 마법 부리기 (따라하기)

로그인을 시도할 때, 무전기(콘솔 화면)에 예쁜 알림이 뜨도록 만들어 볼까요?

1. `TeruTeruServer.Client/TeruClient.cs` 파일을 열어주세요.
2. `LoginAsync` 함수를 찾아서 안으로 들어가 보세요.
3. 함수가 시작되는 바로 아랫줄에, 이렇게 적어보세요!
   ```csharp
   Console.WriteLine($"🚀 삐슝! 관제탑으로 {userId} 님의 로그인 쪽지를 날려 보냅니다!");
   ```
4. 저장하고 나면, 게임에서 로그인을 누를 때마다 저렇게 귀여운 메시지가 무전기에 뜰 거예요! 🎉

---

## ⚠️ 주의사항

무전기를 사용할 때 꼭 기억해야 할 점이 있어요!
관제탑의 주소(**IP 주소**)나 문 번호(**Port 번호**)를 마음대로 바꾸면 안 돼요. 만약 바꾸게 되면, 무전기가 허공에 대고 소리를 치는 꼴이 되어서 영영 놀이공원에 들어갈 수 없게 된답니다. 주파수는 항상 관제탑과 똑같이 맞춰주세요! 📻
