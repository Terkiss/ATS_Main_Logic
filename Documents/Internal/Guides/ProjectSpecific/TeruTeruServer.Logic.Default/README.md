# 🎢 TeruTeruServer.Logic.Default (놀이기구 운영 매뉴얼)

안녕하세요! 이곳은 우리 놀이공원의 진짜 재미가 숨어있는 **'놀이기구 운영 매뉴얼'** 방이에요. ✨

## 🎈 이곳은 무엇을 하는 곳인가요?
관제탑이 손님들을 확인하고 들여보내 주면, 손님들은 본격적으로 게임(놀이기구)을 즐기게 되겠죠?
이곳에서는 "손님이 총을 쏘면 어떻게 해야 할까?", "채팅을 치면 다른 사람들에게 어떻게 보여줄까?" 같은 **진짜 게임 내용(로직)**들을 처리해요. 관제탑에서 넘겨준 쪽지를 읽고 알맞은 답장을 써주는 곳이랍니다.

## 🗺️ 이 방의 지도 (폴더 구조)
이 방에는 놀이기구를 안전하게 돌리기 위한 매뉴얼이 들어있어요.

* 📜 **`LogicPlugin.cs`**: 놀이기구 메인 운영 매뉴얼이에요! "손님이 A를 달라고 하면 B를 줘라!" 같은 모든 규칙이 이 파일 안에 꼼꼼하게 적혀 있답니다.

이제 매뉴얼을 살짝 고쳐서 손님들에게 깜짝 이벤트를 해주고 싶지 않나요?
다음 가이드인 `how-to-code.md`를 열어서 로직 마법사가 되어 보세요! 🚀
