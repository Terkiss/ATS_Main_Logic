# 📜 TeruTeruServer.Commands (마법 주문서 모음집)

안녕하세요! 이곳은 놀이공원 운영자(관리자)가 안내 방송 마이크(터미널)에서 사용할 수 있는 **'마법 주문서'**들이 모여있는 곳이에요. ✨

## 🎈 이곳은 무엇을 하는 곳인가요?
놀이공원을 관리하다 보면 까만 화면(CLI)에 명령어를 쳐서 조종해야 할 때가 있어요. 
예를 들어 화면에 `exit`라고 치면 서버가 안전하게 꺼진다거나, `queuecount`라고 치면 대기열에 몇 명이 있는지 알려주는 식이죠!
이 폴더에는 그렇게 터미널에 입력할 수 있는 명령어(마법 주문)들이 낱장으로 잘 정리되어 있답니다.

## 🗺️ 이 방의 지도 (폴더 구조)
이 방에는 여러 가지 마법 주문서와 그 주문서를 관리하는 직원이 있어요.

* 🧙‍♂️ **`CommandHandler.cs`**: 마법 주문서 관리자예요! 우리가 화면에 무언가 타자기를 치면, "어디 보자... 아, 이건 'exit' 주문이네! 서버 끄는 담당자 나와!" 하고 알맞은 주문서를 찾아줘요.
* 📖 **`ICommand.cs`**: 마법 주문을 만들기 위한 **'기본 양식'**이에요. "모든 마법 주문서에는 반드시 주문 이름과 실행 방법이 적혀 있어야 해!" 라는 규칙을 정해둔 종이랍니다.
* ✨ **`ExitCommand.cs`, `QueueCountCommand.cs` 등**: 실제로 작동하는 개별 마법 주문서들이에요. 이름만 봐도 어떤 마법이 나갈지 딱 알겠죠?
