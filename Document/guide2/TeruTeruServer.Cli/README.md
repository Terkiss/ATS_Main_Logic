# 🎤 TeruTeruServer.Cli (놀이공원 전원 스위치)

안녕하세요! 이곳은 우리 놀이공원의 하루를 시작하는 **'전원 스위치'**가 있는 방이에요. ✨

## 🎈 이곳은 무엇을 하는 곳인가요?
아무리 거대하고 멋진 놀이공원이라도 누군가 아침에 와서 "전원 켜기!" 버튼을 누르지 않으면 멈춰 있겠죠?
이곳은 프로그램이 가장 먼저 실행되는 출발점이에요. 놀이공원의 기본 설정(최대 손님 수 등)이 적힌 쪽지를 읽어오고, 각 부서의 직원들에게 일을 배정해준 뒤, 최종적으로 "자, 영업 시작합니다!" 하고 방송을 하는 역할을 해요.

## 🗺️ 이 방의 지도 (폴더 구조)
이 방에는 놀이공원의 아침을 여는 중요한 파일들이 있어요.

* 💡 **`Program.cs`**: 놀이공원의 가장 큰 메인 전원 버튼이에요. 여기서부터 모든 코드가 시작된답니다!
* 📋 **`ConfigManager.cs`**: `config.txt`라는 쪽지를 읽어서, 오늘 놀이공원에 손님을 몇 명이나 받을지 설정해주는 친절한 비서예요.

자, 전원 스위치를 누를 때 멋진 안내 방송이 나오도록 만들어 볼까요?
다음 가이드인 `how-to-code.md`를 열어보세요! 🚀
