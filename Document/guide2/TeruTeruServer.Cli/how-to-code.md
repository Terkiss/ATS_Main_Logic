# 🛠️ 아침 안내 방송 바꾸기 (코드 수정법)

안녕하세요! 이곳에서는 놀이공원 전원 버튼인 `Program.cs` 파일을 열어서 직접 코드를 수정해 볼 거예요. 
서버가 켜질 때 멋진 인사말이 나오도록 만들어 볼까요? ✨

## 🧐 가장 중요한 코드 읽어보기

놀이공원의 아침을 여는 파일은 `TeruTeruServer.Cli/Program.cs` 예요.

### 1. 진짜 전원 스위치 (`Main` 함수)
파일의 **14번째 줄**을 보면 `static void Main(string[] args)` 라는 부분이 있어요.
모든 C# 프로그램은 이 `Main` 이라는 이름의 문을 열고 들어가면서 시작돼요! 
안으로 들어가면 설정 파일(`config.txt`)을 읽고, 직원들을 배치(`ConfigureServices`)하는 일들이 차례대로 일어납니다.

### 2. 안내 방송을 하는 곳
파일의 **29번째 줄** 쯤을 보면 서버가 켜졌다는 것을 알려주는 코드가 있어요.
```csharp
Console.WriteLine("=== TeruTeruServer AI Engine Runtime Started ===");
```
이곳이 바로 관리자 화면(까만 창)에 글씨를 띄워주는 마이크 같은 곳이에요!

## 🪄 나만의 마법 부리기 (따라하기)

딱딱한 영어 방송 대신, 우리만의 활기찬 한국어 인사말로 바꿔볼까요?

1. `TeruTeruServer.Cli/Program.cs` 파일을 열어주세요.
2. **29번째 줄** 쯤에 있는 `Console.WriteLine` 코드를 찾아보세요.
3. 괄호 안의 쌍따옴표(`""`) 사이에 있는 글자를 이렇게 예쁘게 바꿔주세요!
   ```csharp
   Console.WriteLine("🎡 삐빅- 환상적인 놀이공원, 오늘 영업을 시작합니다! 🎡");
   ```
4. 저장하고 서버를 켜보면, 까만 화면에 귀여운 관람차 이모티콘과 함께 여러분이 적은 인사말이 뜰 거예요! 🎉

## ⚠️ 주의사항

**36번째 줄**부터 시작하는 `ConfigureServices` 부분은 놀이공원 부서들끼리의 **'비상 연락망'**을 연결해두는 아주아주 예민한 작업실이에요! 
여기서 줄을 하나라도 잘못 끊거나 연결하면 직원들끼리 연락이 안 돼서 놀이공원이 마비되어 버리니, 이 부분은 조심해서 구경만 해주세요! 🤫
